隐私政策

更新日期：2026-03-04

欢迎使用《电子斗蛐蛐》。本页面说明我们如何处理与保护你的信息。

1. 我们收集的信息

本游戏为纯前端单机网页游戏，默认不要求注册账号，不主动收集姓名、手机号、身份证号等敏感个人信息。

网站托管平台或广告服务商可能会收集基础访问数据（如设备类型、浏览器信息、IP 地址、访问时间）用于统计与安全防护。

2. Cookie 与广告

网站可能接入第三方广告服务（例如 Google AdSense）。广告服务商可能使用 Cookie 或类似技术，用于展示更相关广告、衡量广告效果、防止欺诈。

你可以在浏览器设置中管理或禁用 Cookie，但这可能影响部分功能体验。

3. 信息用途

相关信息仅用于：

  • 保障网站与游戏功能正常运行
  • 分析访问情况，优化产品体验
  • 广告展示与结算（若开启广告）

4. 第三方服务

本网站可能使用以下第三方服务：

  • GitHub Pages（网站托管）
  • Google AdSense（广告服务，可选）

第三方服务会根据其自身隐私政策处理数据。建议你同时阅读第三方隐私条款。

5. 未成年人说明

如你是未成年人，请在监护人指导下访问和使用本游戏。

6. 联系我们

如果你对本隐私政策有疑问，可通过 GitHub 仓库 Issue 联系：

https://github.com/gujijia12/game/issues
